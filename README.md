# Tempo

A simple calendar, todo list, note taking web application.


## Technologies Used
* JavaScript (ES5, ES6)
* HTML5
* CSS3
* React
* Bootstrap 4
* Node.js
* Firebase
* Chakra UI

## Features
* User can login
* User can sign up
* User can view schedule
* User can add new notes on selected date
* User can delete notes on selected date
